# AyuubProject

A modern School Management System web application for Al-Qalam PSS, designed to streamline administration, student records, results, communication, and more.

## Features
- Admin dashboard with sidebar navigation
- Student, teacher, and user management
- Class and fee management
- Event calendar
- Library management
- News & events, content management
- Secure login for admin, teachers, students, and parents
- Responsive design using Bootstrap 5

## Folder Structure
```
ayuubproject/
├── css/                # Custom stylesheets
├── images/             # Project images
├── index.html          # Home page
├── admin.html          # Admin dashboard
├── student.html        # Student records
├── teacher.html        # Teacher management
├── ...                 # Other HTML pages (see file list)
├── mai.js              # Main JavaScript file
└── README.md           # Project documentation
```

## Setup Instructions
1. Clone the repository:
   ```sh
   git clone https://github.com/Ayoub2046/ayuubproject.git
   ```
2. Open the project folder in your code editor.
3. Open `index.html` in your browser to view the site locally.
4. To deploy online, use GitHub Pages or any static hosting provider.
